# Abey Games – Developer Website

This repository hosts the official developer website for **Abey Games**.

It is primarily used to support **AdMob app-ads.txt verification** and establish ownership of published mobile games.

---

## 📄 app-ads.txt

The `app-ads.txt` file is used to authorize digital advertising systems to sell ad inventory for our apps.

This helps:
- Improve ad revenue
- Prevent unauthorized ad usage
- Verify app ownership with ad networks like Google AdMob

---
